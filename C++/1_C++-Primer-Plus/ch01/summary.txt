C++는 C 언어에 객체 지향 프로그래밍과 일반화 프로그래밍 기능을 추가한 언어이다.
C++는 세 가지 프로그래밍 방식-C의 절차적 언어 방식, C++의 객체 지향 언어 방식, C++ 템플릿의 일반화 프로그래밍 방식-을 하나로 결합한 다중 패러다임 언어이다.

일반적으로 컴퓨터 언어는 데이터와 알고리즘이라는 두 가지 개념을 다룬다.
절차적 프로그래밍과 일반화 프로그래밍은 알고리즘 측면을 강조하지만, 객체 지향 프로그래밍은 데이터 측면을 강조한다.
-. 절차적 프로그래밍 : 알고리즘 측면을 강조하며, 하향식(top-down) 설계 방식을 가진다.
-. 일반화 프로그래밍 : 알고리즘 측면을 강조하며, 데이터형에 무관한 코드를 작성하겠다는 개념이다.
-. 객체 지향 프로그래밍 : 데이터 측면을 강조하며, 상향식(botton-up) 설계 방식을 가진다. 해결해야 할 문제의 특성에 맞게, 데이터형(클래스) 자체를 설계하겠다는 개념이다.
